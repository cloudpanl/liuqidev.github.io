xxx.github.io连接已重置多在于没有使用https。

修改hosts文件（视窗操作系统一般在系统盘的C:\Windows\System32\drivers\etc路径下），

添加如下解析主机：

依然还是不行。

查明原因是没有使用https网络协议所致。

使用谷歌浏览器（Chrome）内核，默认都是https。出现问题，可能就是hosts或者解析的事了，其余基于谷歌浏览器内核，对于某些网站，执行以下操作：

  • 在浏览器地址栏输入chrome://net-internals/#hsts
  • 将要访问的网址添加到Input a domain name to add it to the HSTS set:下的域名（Domain）中
  • 点击添加（Add）

之后就能使用https为开头访问了。
