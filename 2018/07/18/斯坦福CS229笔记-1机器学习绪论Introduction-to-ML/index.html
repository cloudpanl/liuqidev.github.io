前言

CS229是吴恩达(Andrew Ng)教授在斯坦福大学教授的一门先修课程，这里是学习其较早的版本所做的笔记，相较于之后的课程，这个版本容量较大，介绍了很多经典的机器学习方法和应用。缺点是有的地方可能已经过时，以及在有些方面介绍得不够全面。因此适合用于了解和学习机器学习的通用和经典方法。在作业和实现上使用的是Matlab和Octave，在学习到这相关方面时可以选择跳过，选择学习Coursera上其较新版本的课程或者微专业配套的作业作为补充。

第一讲笔记

  • Machine Learning

    • What is ML:
      • Field of study that gives computers the ability to learn without being explicitly programmed. - Authur Samuel(1959)
      • “A computer program is said to earn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured P, improves with experience E.” -Tom Mitchell(1998)

  • Supervised Learning

    • Why do we call xxx(房价预测) supervised learning?:

      • “We are supervising the algorithm, in other word, we give the algorithm the quote, right answer, and the we want the algorithm to learn the association between the inputs and the outputs and give us more right answers.”

    • Regression Problem回归问题：

      • the variable we’re trying to predict is continuous.

    • Classification Problem分类问题

      • the variable we’re trying to predict is discreet离散的 rather than continuous.

        • 举例：经典例子，预测乳腺癌是否是恶性

  • Learning Theory

    • 将学习算法应用到具体实际问题之中

  • Unsupervised Learning

    • Clustering聚类问题：
      • 2D图像聚类重建3D模型
      • 天文，市场分析，计算机集群聚类等
      • Cocktail Party Problem: 从多个说话者的声音混合中提取出每个说话者的声音
        • ICA Algorithm:

  • Reinforcement Learning

    • 用于不需要进行一次决策的场合。对正确的决策进行奖励，错误的进行惩罚，以求获得奖励最大。
    • 应用：飞行器自主飞行，机器人训练和学习。
