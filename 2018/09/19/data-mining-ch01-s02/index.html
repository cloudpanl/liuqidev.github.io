聚类（Clustering）

概念

将观察到无标记项集合划分成若干子集（subsets成为clusters簇）,同一个簇中的项具有相似性。

Clustering is the assignment of a set of observations into subsets (called clusters) so that observations in the same cluster are similar in some sense.

  • 异常值（离群点，Outliers）
  • …
